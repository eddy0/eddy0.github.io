技巧

  • 高级版的 log

  • 追加 log 内容到文本

  • Object.assign(a, b) 拼接两个对象的方法, 后面的 b 会加到前面的 a 去, 并且 a 是拼接之后的对象， 如果 a，b 一样, 同名 后面 会覆盖前面的

  • 返回实例的方法就必然要用 static 方法
  • 修改东西一定要先注释，测试没有问题之后再删除, 再右键文件写local history 的label, 会记录删除了什么
  • 不要再一个函数做两个事情, 逻辑就是逻辑, 页面就是页面
  • 事不过三原则, 对象超过两个要换行, 变量调用超过两个要用更简短的变量接住
  • 写 what 不写how, 疯狂拆函数, 一个函数一件事情

RESTFUL 与 路由方案

  1. RESTFUL
    把 path 当做资源, http 各种方法当做动词来操作这些资源.

    app.get(‘/todo/:id’)
    app.post(‘/todo’)
    app.put(‘/todo’)
    app.patch(‘/todo’)
    app.delete(‘/todo’)
