重写 toString

原因: object.toString() 返回的是 [object Object]
当我们发送 html 的时候, 需要发送字符串的格式 所以要把数据里面的对象转化为字符串
用 String.replace() 的时候， 会自动先调用 toString() 方法，如果替换的是对象, 就会变成上面的格式

重写就是用 JSON 把对象改成字符串形式, 给类添加一个 toString(), 覆盖掉默认的 toString()
这样使用 replace 的时候, 对象就可以变成一个可以显示的字符串了

http 组成

  • protocol: http, https
  • host:
  • port: 80(http), 443(https)
  • path: /

dns 服务器 —- domain name server

http 协议: 超文本传输协议

  1. 服务器按规定的格式发送给文本数据(请求)到服务器
  2. 服务器解析请求, 按照规定的格式返回文本数据(响应到浏览器)到浏览器
  3. 浏览器解析得到的数据， 并做相应的处理

  1. 建立连接
  2. 监听连接
  3. 接受数据
  4. 处理数据
  5. 发送数据
  6. 关闭连接

  1. 配置相关参数
